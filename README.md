# portfolio_frontend

------PORTFOLIO------
..En este proyecto realizaremos una pagina web en donde se podra ver el portfolio del autor del sitio, tanto como sus aportes y proyectos realizados. Tambien incluiremos un apartado donde los desarrolladores que quieran sumarse, podran crearse una cuenta con el fin de realizar aportes, brindar ayuda o publicar su contenido.


Repositorio que utilizaremos para desarrollar el frontend de nuestra pagina web

REGISTRO DE CAMBIOS:
*Hemos agregado un archivo llamado login.html donde haremos que los desarrolladores que quieran realizar sus aportes puedan logearse con su cuenta para asi, poder modificar, agregar o eliminar contenido que hayan subido dentro de la web
*Hemos agregado un archivo llamado foro.html donde alli la comunidad podra ver los posteos realizados por los demas desarrolladores, publicaciones nuevas del creador del sitio y noticias de interes
*Hemos agregado los respectivos botones para que de cierto modo, estemos en cualquier parte del sitio, podamos acceder a las demas paginas creadas dentro del mismo.
